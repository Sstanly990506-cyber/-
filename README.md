# 上光廠同步網站系統

這版改為「可多人跨電腦共同使用且同步」的網站架構，流程如下：

1. 開啟網站先選擇公司
2. 輸入帳號密碼登入
3. 登入後先看到三個導航方框（不是直接顯示內容）：
   - 1 工單作業系統
   - 2 客戶系統
   - 3 財金系統
4. 點擊方框才會進入該系統內容頁，並可返回導航頁

## 同步方式

- 資料儲存在伺服器 `db.json`（不是瀏覽器 localStorage）。
- 同公司使用者透過 SSE 即時同步（任一台新增/刪除，其他台會更新）。

## 預設登入

- 公司：`上光廠 A 公司` / `上光廠 B 公司` / `上光廠 C 公司`
- 帳號：`admin`
- 密碼：`123456`

## 啟動

```bash
npm start
```

啟動後開啟：`http://localhost:4173`
