# 故障排除

## `ERR_CONNECTION_REFUSED`

代表瀏覽器找不到正在執行的伺服器。確認伺服器視窗仍開啟，並使用啟動畫面顯示的實際連接埠。

## 手機無法連線

- 手機不可使用 `127.0.0.1`，該位址代表手機本身。
- 手機與電腦必須連接同一個 Wi-Fi。
- 伺服器需使用 `--host 0.0.0.0` 啟動。
- 確認防火牆允許私人網路連線。

## 資料沒有同步

- 確認所有裝置使用相同的主機網址。
- 檢查頁面上的同步狀態。
- 開啟 `/api/health` 確認伺服器可用。
- `file://` 與純靜態模式只會使用各瀏覽器自己的資料。

## 畫面沒有更新

按 `Ctrl + F5` 強制重新整理，並確認目前資料夾與伺服器是最新版本。
